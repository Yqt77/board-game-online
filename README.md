# 三棋联机对战

一个本地可运行的网页棋类游戏，支持：

- 五子棋
- 象棋
- 围棋
- 两人联机同房间对战

## 启动本地服务

```bash
node server.js
```

然后打开：

```text
http://127.0.0.1:3000
```

## 让外网访问

如果你现在只有本地访问，没有公网 IP，推荐先用 Cloudflare Tunnel 的 quick tunnel。

大致步骤是：

1. 安装 `cloudflared`
2. 保持本地服务运行在 `127.0.0.1:3000`
3. 执行：

```bash
cloudflared tunnel --url http://127.0.0.1:3000
```

4. 它会输出一个 `trycloudflare.com` 的公网地址
5. 把这个公网地址填到页面里的“公网地址”输入框
6. 点击“复制房间链接”，把链接发给对方

这个项目已经做了轮询兜底，所以即使你的隧道入口不支持 SSE，房间同步也能工作。

### Windows 一键脚本

如果你在 Windows 上运行，可以直接用：

```powershell
.\scripts\install-cloudflared.ps1
.\scripts\start-public.ps1
```

`start-public.ps1` 会在必要时先启动本地服务，再打开 Cloudflare quick tunnel。

## 房间分享

1. 创建一个房间。
2. 如果你用的是公网域名或内网穿透地址，在页面里填到“公网地址”输入框。
3. 点击“复制房间链接”，把链接发给对方。

如果你直接通过公网地址打开这个站点，链接会自动跟着当前域名走，不需要额外配置。

## 规则概览

- 五子棋：连成五子获胜。
- 象棋：实现了主要走子规则、吃子、自身将帅不能送吃，以及将帅对面判定。
- 围棋：实现了落子、提子、禁自杀、简易打劫、双停计分。

## 说明

- 服务端是无依赖的 Node.js 原生 HTTP 实现。
- 房间状态保存在内存中，重启服务后会清空。
